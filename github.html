『 ☆Why use GitHub for Website Maintenance?☆ 』
GitHub is an excellent tool for website maintenance because it allows for version control, collaboration, and easy deployment.
With GitHub, you can track changes made to your website over time, collaborate with others on the same project, and deploy your website to a live server quickly and easily.
『 ☆Sign Up For Github☆ 』

You can click the box above to get to GitHub's sign up page!

Then, fill in the account creation form with your username, email address, and a strong password. You can also choose to sign up using your existing Google or Microsoft account.

You can now explore GitHub and start using its features, such as creating repositories, collaborating with others, and contributing to open-source projects.
『 ☆Create a Git repository☆ 』
Once you have a GitHub account, create a new repository by clicking the "New" button on your GitHub homepage. Give your repository a name and description that describes your website.

『 ☆Upload your website files☆ 』
To upload your website files, drag and drop them into the repository or use the upload button in the web interface.

Be sure to include an index.html file to serve as the homepage of your website.
『 ☆Host your website☆ 』
To host your website on GitHub, go to the "Settings" tab of your repository and scroll down to the "GitHub Pages" section. Choose a source for your website (e.g. the master branch) and click "Save". Your website will now be live at username.github.io/repository-name.

『 ☆Install GitHub Desktop☆ 』
GitHub Desktop is an application that enables you to interact with GitHub using a GUI instead of the command line or a web browser. GitHub Desktop encourages you and your team to collaborate using best practices with Git and GitHub.

『 ☆Clone your repository☆ 』
Open GitHub Desktop and click on the "Clone a repository" button. Select the repository you want to work on from the list of repositories on your GitHub account. This will create a local copy of your repository on your computer.

『 ☆Open your repository in a code editor☆ 』
To make changes to your website, open your repository in a code editor of your choice. Edit the files in the repository as needed.

『 ☆Commit changes☆ 』
Once you've made changes to your website, commit them to the repository with a meaningful commit message. You can do this in GitHub Desktop by selecting the files you want to commit and entering a commit message.

『 ☆Push changes to GitHub☆ 』
To publish your changes, click on the "Push origin" button in GitHub Desktop. This will push your changes to the remote repository on GitHub.
