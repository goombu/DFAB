『 ☆Creation Of Basic Site Content☆ 』

To start, I tried to highlight what were some necessary elements to have in my site.

Introduction
I intended for this page to encapsulate my site's intention through simple phrases and visuals. This could also provide some simple information to orient users, such as a basic directory or explanation to the site's purpose. I included a calendar here to also give a better sense of understanding.

Navigation Bar
This will consist of various contents of the site that the user can access through a simple menu. This would contain primary subjects such as my "About Me" and my "Final Project". There would be deeper documentation for each element that could be accessed here

Exercises
I had originally intended to put this under the navigation menu, however I realised that there were many informative exercises that we would have, and to better document my journey in DFAB I felt that a separate menu to properly display the individual lessons learnt would be more accessible for interested users.

『 ☆Creation Of Project Site☆ 』

I started off by coding the CSS to add a unique style of the website, trying to integrate elements popular in early 2000s website, which I was inspired by. For example, majority of the site is written using font MS Gothic, which was a popular font used during y2k.

I then started editing the navigation table with html, allocating specific links and titles for each section. I used a pastel rainbow colour palette for this, while still maintaining the retro vibe of the site.

I began to to create the bried introduction of the website, using a blurring effect on the container to draw more attention to these basic details.

Here I added a link to a calendar code I had referenced, and added the basic website information. This was to set users in the right mindset before exploring the site, so that they can better understand the intended theme and purpose.

I added another navigation bar for the exercises. Instead of simple texts, I custom made blinkies, a common staple in old websites. These helped draw more attention to the contents in contrast to the white background, while also adding some fun elements to the site theme.

Finally, I added some aesthetic embellishments to further ennsure the execution of the site them. I added a pagedoll of Espresso cookie(a character from a game i like) as a representation of one of my favourite characters, and special effects for some texts using custom blinkie codes I made myself.

To add a finishing aesthetic embellishment, I searched for tutorials on how to add effects to the cursor. Using a Javascript template I found online, I added in star effects to the mouse so that gave an unique vibe to the home page.
